# 产品交互设计

## 一、产品定位

**核心价值主张：** 不是告诉你「买谁」，而是告诉你「这场比赛的概率是多少，以及和赔率相比是否值得」。

**目标用户：**
- 主要：对足球有一定了解，希望用数据辅助判断的中度用户
- 次要：纯粹对足球数据/AI感兴趣的爱好者

---

## 二、产品形态选择

### 方案 A：投注面板式（直接显示型）

```
┌─────────────────────────────────────────┐
│  中超 第22轮  2026-04-05 19:35          │
│                                         │
│  上海海港  vs  北京国安                  │
│                                         │
│  ████████░░░░░  ░░░░░░░  ░░░░░░░░      │
│    主胜 52%       平 24%    客胜 24%    │
│                                         │
│  体彩赔率: 2.1 / 3.2 / 3.5             │
│  ★ 主胜有价值 (+8% vs 赔率)             │
└─────────────────────────────────────────┘
```

**优点：** 信息密度高，决策快，适合比赛日快速浏览
**缺点：** 容易被误解为直接推荐「买这个」，合规风险

### 方案 B：问答式（AI 对话型）

```
用户: 今晚海港 vs 国安怎么看？

AI: 根据模型分析：

📊 概率预测：
  主胜（海港）52% | 平局 24% | 客胜 24%

📈 近期状态：
  海港近5场：3胜1平1负，主场4连胜
  国安近5场：1胜2平2负，客场低迷

💡 数据提示：
  主赔2.1对应隐含概率47.6%，
  模型给出52%，存在约4%的正期望差。

⚠️ 足球存在高随机性，任何预测均不保证结果。
```

**优点：** 自然，有解释性，规避「直接推荐投注」的灰色地带
**缺点：** 每场都对话略繁琐，不适合同时看多场

### 方案 C：混合型（推荐）

- **首页/列表**：投注面板式，显示当日所有比赛的简洁概率
- **详情页**：点击进入，自动生成 AI 解读（相当于方案 B 的输出）
- **问答入口**：底部悬浮，支持自然语言提问

## 三、推荐：混合型产品结构

### 3.1 页面结构

```
首页
  ├── 今日赛事列表（卡片式，显示三项概率 + 价值信号）
  ├── 联赛筛选（中超 / 中甲 / 亚冠 / 其他）
  └── 快速问答入口

比赛详情页
  ├── 概率分布（可视化）
  ├── AI 解读（自动生成）
  ├── 近期状态对比（折线图）
  ├── 历史交锋记录
  └── 数据说明（模型准确率、校准曲线）

数据透明页
  ├── 模型在各联赛的历史 RPS
  ├── 校准曲线
  └── 方法论说明（白话版）

问答页（可选 V2）
  └── 自然语言提问，AI 实时分析
```

### 3.2 关键 UI 决策

**概率展示：用横向进度条，不用饼图**

饼图让人觉得「大块的那个就是答案」，进度条更中性，传达「不确定性」的感觉。

```
主胜  ████████████░░░░░░░░  52%
平局  ██████░░░░░░░░░░░░░░  24%
客胜  ██████░░░░░░░░░░░░░░  24%
```

**价值信号：用「+ / - / ~」三档，不用数字**

```
✦ 有价值  (模型概率 > 赔率隐含概率 + 阈值)
~ 中性    (接近)
✗ 无价值  (模型概率 < 赔率隐含概率)
```

**强制显示不确定性提示：**

每个预测下方固定显示：「足球存在高随机性，预测仅供参考，不构成投注建议」

## 四、移动端优先

国内用户以手机访问为主，建议：

- **第一阶段**：移动端优先的 Web（H5），使用 Next.js + Tailwind 响应式
- **第二阶段**：微信小程序（复用后端 API，重写前端）

微信小程序的优势：分享传播、消息推送（赛前提醒）、无需安装。

## 五、用户旅程

```
场景：用户想了解今晚中超比赛

1. 打开首页
   → 看到今日 3 场中超比赛卡片
   → 每张卡片显示：球队名、时间、三项概率条、价值信号

2. 点击感兴趣的比赛
   → 进入详情页
   → 看到 AI 自动生成的简短解读（3-4 句话）
   → 可以展开看近期状态、历史交锋

3. 想问具体问题
   → 点击「问 AI」
   → 输入「海港今晚阵容有什么变化」
   → AI 结合阵容数据回答

4. 比赛结束后
   → 该卡片更新为实际结果
   → 模型概率保留（可以看预测是否合理）
```

## 六、合规考量

国内运营足球预测产品的灰色地带：

- **规避**：不直接说「买主胜」，不提具体彩票产品名
- **措辞**：用「概率分析」「数据参考」代替「预测」「推荐」
- **免责**：每页显示明确免责声明
- **定位**：体育数据分析工具，而非彩票辅助工具

如果未来考虑商业化：
- 数据订阅（按联赛/赛季）
- API 开放给开发者
- 避免直接接入彩票购买流程
