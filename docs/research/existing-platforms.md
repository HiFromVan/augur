# 现有平台分析

## 一、学术透明型（值得参考）

### Kickoff.ai
- **网址**：kickoff.ai
- **背景**：EPFL（洛桑联邦理工）研究组产品，Lucas Maystre 等人
- **方法**：贝叶斯配对比较模型 + 球员核函数（Player Kernel）
  - 不把球队作为整体，而是建模为球员集合
  - 球员技能用高斯过程动态追踪随时间演化
  - 输出概率分布，用 log-loss 评估
- **开源**：引擎开源 → [lucasmaystre/kickscore](https://github.com/lucasmaystre/kickscore)
- **适用**：学术参考、学习方法论
- **局限**：面向欧洲联赛，无中文赛事支持

### MyGameOdds
- **网址**：mygameodds.com
- **特点**：已分析 45000+ 场比赛，生成 200万+ 市场预测
- **透明度**：公开显示每个联赛的滚动 100 场准确率，可独立验证
- **开源**：闭源
- **价值**：提供了「如何展示准确率」的产品设计参考

## 二、商业预测平台

### Footixify
- **网址**：footixify.com
- **开源**：是，[Mg30/footixify](https://github.com/Mg30/footixify)
- **模型**：仅使用球员统计特征构建 ML 概率，附带价值投注扫描器（对比庄家赔率）
- **产品形态**：Web app，显示各场比赛三项概率 + 是否存在价值投注机会
- **局限**：欧洲数据，国内赛事数据未接入

### NerdyTips
- **网址**：nerdytips.com
- **方法**：深度神经网络 + 模式识别，处理比赛历史、天气、伤病、xG 等数十个变量
- **覆盖**：160+ 联赛，已分析 234157 场
- **开源**：闭源
- **商业模式**：订阅制，但免费版有基础预测

### FootBot
- **覆盖**：英超、西甲、意甲、德甲、欧冠
- **市场**：胜平负、大小球、双方进球
- **价格**：约 $10/月
- **开源**：闭源

### FiveThirtyEight SPI（已停更）
- **模型**：Soccer Power Index，球队进攻/防守能力综合评分
- **数据**：[GitHub 公开](https://github.com/fivethirtyeight/data/tree/master/soccer-spi)，历史预测数据可下载
- **价值**：数据集可用于模型训练和基准对比

## 三、国内现状

### 体彩官方
- 彩票中心不提供预测，只提供赔率和历史数据
- 无公开 API

### 第三方分析网站
- **雷速体育**（leisu.com）：提供历史数据、伤病、阵容，但无预测
- **球探网**（jczq.com）：提供亚盘欧赔走势，庄家动向分析
- **500彩票**（500.com）：提供赔率对比，无 ML 预测
- **WhoScored**（中文版）：详细球员统计，可作为特征数据来源

## 四、关键差异：欧洲 vs 国内市场

| 维度 | 欧洲平台 | 国内需求 |
|------|----------|----------|
| 数据来源 | Bet365/Pinnacle 赔率 | 体彩北单/竞彩 |
| 玩法 | 3way（胜平负）、亚盘 | 北单（固定赔率）、竞彩（浮动赔率） |
| 联赛 | 英超、西甲为主 | 中超、中甲 + 世界杯/欧洲杯期间热点联赛 |
| 数据质量 | 完善 | 中超数据相对匮乏，尤其球员层面 |
| 监管 | 博彩合法 | 灰色地带，需规避直接博彩导流 |

## 五、产品设计启发

从现有平台观察到的最佳实践：

1. **透明度是信任基础** — MyGameOdds 公开每联赛滚动准确率，比声称"83%准确率"可信得多
2. **概率而非结论** — 好的产品给概率分布，而不是直接说"买主胜"
3. **价值投注视角** — Footixify 的核心价值不是预测对错，是找到模型概率 > 庄家隐含概率的机会
4. **解释性很重要** — 用户需要知道「为什么」，不只是「是什么」
